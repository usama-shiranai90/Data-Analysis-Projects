# Data Insights Hub

This repository is a comprehensive data analysis project consisting of three major studies:
1. **A Visual History of Nobel Prize Winners**
2. **Analyzing Crime in Los Angeles**
3. **Exploring NYC Public School Test Result Scores**

Each component aims to provide detailed insights through data visualization and statistical analysis, catering to diverse interests from historical achievements to urban safety and educational performance.

## Projects Overview

### A Visual History of Nobel Prize Winners

This project visualizes the history and trends associated with Nobel Prize winners over the years. It includes interactive charts and graphs representing demographics, fields of contribution, and other interesting statistics.

### Analyzing Crime in Los Angeles

Focused on crime statistics in Los Angeles, this analysis seeks to identify trends, hot spots, and the effectiveness of law enforcement strategies over recent years. The project uses public crime data to create maps and trend analyses.

### Exploring NYC Public School Test Result Scores

This component examines the performance of public schools in New York City by analyzing standardized test results. The aim is to highlight educational disparities and success stories within the NYC school system.

## Built With

- Python (for data analysis and visualization)
- Jupyter Notebooks (for interactive data exploration)
- Pandas & NumPy (for data manipulation)
- Matplotlib & Seaborn (for creating visualizations)
- Folium (for geospatial analysis)
